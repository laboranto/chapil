Surface

배경면 레이어를 정의하는 토큰. 페이지 배경, 카드, 패널의 시각적 깊이를 표현한다.

보조 배경면. 입력창, 코드 블록, 구분 영역.

Light
Dark
--surface-elevated

카드·바텀시트·모달. 반드시 --blur와 세트.

--surface-fade
Light: rgba(255,255,255,0) Dark: rgba(21,21,21,0)

그라디언트 끝점 전용. topbar·bottom-bg의 linear-gradient 종단점.

✅ Do
카드 → surface-elevated + blur + shadow

--surface-elevated를 배경으로, --blur와 --shadow 함께 사용.

❌ Don't
페이지 배경에 surface-elevated → 반투명 중첩

페이지·body 배경에 --surface-elevated 금지. 반투명 중첩 시 색상이 뭉침.

Outline

구분선과 테두리 토큰. 두 단계로 시각 계층을 표현한다. 두 토큰을 혼용하지 않는다.

미리보기
--outline
Light: 1px solid rgba(128,128,128,0.25) Dark: 1px solid rgba(96,96,96,0.25)

카드 테두리, 목록 구분선, 섹션 경계.

미리보기
--outline-strong
Light: 1px solid rgba(128,128,128,0.75) Dark: 1px solid rgba(96,96,96,1)

강조 테두리. 버튼(.btn), 입력창 포커스.

✅ Do
목록 카드 → outline

목록·카드 --outline, 버튼·입력 강조 --outline-strong.

❌ Don't
목록 카드에 outline-strong

혼용 시 시각 계층이 평탄해져 UI 구조가 읽히지 않음.

Elevation

그림자와 블러로 레이어의 높이를 표현한다. --blur는 반드시 --surface-elevated와 세트로 사용한다.

--blur
blur(2.5px)

backdrop-filter 전용값. glassmorphism 효과. --surface-elevated 없이 단독 사용 금지.

--shadow
--shadow
0 0 5px 0 rgba(0,0,0,0.15)

기본 상태 그림자.

--shadow-hover
--shadow-hover
0 0 10px 0 rgba(0,0,0,0.2)

hover 상태 그림자. shadow보다 넓게 퍼짐.

--shadow-active
--shadow-active
0 0 5px 0 rgba(0,0,0,0.25)

active 상태 그림자. shadow보다 강도 높고 범위 좁음.

✅ Do
surface-elevated + blur + shadow

--blur는 --surface-elevated와 반드시 세트로 사용.

❌ Don't
blur만 적용 → 뒤 내용이 비침

--blur 단독 사용 금지. 면(surface-elevated) 없이 블러만 적용하면 뒤 콘텐츠가 비쳐 가독성 손상.

Shape

모서리 반경 토큰. 컴포넌트 크기와 역할에 따라 두 단계로 구분한다.

radius-sm
--radius-sm
10px

입력창, 작은 카드, 드롭다운, 배지.

radius-lg
--radius-lg
32px

바텀탭, pill 버튼, 토스트 알림, 온보딩 버튼.

✅ Do

입력창 --radius-sm, pill 버튼 --radius-lg.

❌ Don't

컴포넌트 그룹 내 반경 혼용 금지. 시각 언어 충돌.

Layout

레이아웃 크기와 iOS safe area 토큰. 직접 수정하지 않고 참조만 한다.

--size-nav
64px

바텀 내비게이션 높이. content의 padding-bottom 계산에 사용.

--size-layout
600px

콘텐츠 최대 너비. max-width에 적용해 태블릿·데스크탑 레이아웃을 제한.

--safe-top
env(safe-area-inset-top, 0px)

iOS 노치·Dynamic Island 상단 여백. topbar padding에 사용.

--safe-bottom
env(safe-area-inset-bottom, 0px)

iOS 홈 인디케이터 하단 여백. bottom-nav margin에 사용.
